Aliasing

这个名字很不好理解，啥是aliasing呢，翻译是别名。我的理解就是，一个东西有多种名字。说大白话就是，当我在操作object的时候，比如我让两个obejct相等(assignment)，我不是在让这两个object的值相等，我是在让他们指向同一个物体。
那这个物体就有了多个名字可以叫。换句话说就是，两个object 的reference被我用于指着同一个东西。所以在之后其中一个reference对物体做啥，另一个reference也会受到影响，为啥？因为它页指着同一个物体呀。

Random (seed)

在学Mathematical operators的时候遇到了Random，顺便做了些研究
random并不是真的随机，其实所谓的随机数都是follow某些函数算出来的，所以seed种子值是用来给函数一个起始值。
由此可知，如果两个random object用的是同一个种子值，那么他们所产生的随机数也就会是一样的。
