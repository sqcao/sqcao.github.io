JavaScript 学习总结

#JavaScript variable
If you initiate a undefined variable,it will be automatically set as global variable.

for example carname = “Volvo”
this will be a global variable

##JavaScript variable life time

local variable will vanish after the function
global variable will vanish after the page is closed

#Operator

##Comparison operator

全等（===）, for example x=5;
x === 5 return true
But x === “5” return false

##Condition operator

Syntax
variableName = (condition)?value1:value2
example:
greeting = (visitor == “AMOR”)?”Hello Amor!”:”Who are you!”
If the visitor is amor then greet, if not then warning
